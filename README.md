# 🏋️‍♂️ Academia Fitness - Website Responsivo

Um projeto de interface moderna e dinâmica desenvolvido para o setor de fitness. O objetivo foi criar uma landing page atraente que destaca os serviços, planos e a infraestrutura de uma academia, com foco total na conversão de novos alunos.

🔗 **[Clique aqui para ver o projeto online](https://davizzjava.github.io/academia-fitness/)**

---

## 🛠️ Tecnologias e Conceitos

Para este projeto, foquei em técnicas que garantem uma navegação fluida e um visual profissional:

* **HTML5 Semântico:** Melhor estruturação para SEO e acessibilidade.
* **CSS3 Avançado:** Uso de variáveis, transições suaves e efeitos de hover.
* **Flexbox & Grid Layout:** Para garantir que o site seja 100% responsivo (funciona em celulares, tablets e desktops).
* **UI/UX Design:** Escolha de cores e tipografia voltadas para energia e saúde (nicho fitness).

---

## 🌟 Diferenciais do Projeto

- **Mobile First:** A interface foi pensada para quem busca informações rápidas pelo celular.
- **Seção de Planos:** Tabela de preços clara e organizada para facilitar a decisão do usuário.
- **Otimização de Imagens:** Carregamento rápido para não perder visitantes.
- **Chamadas para Ação (CTA):** Botões estrategicamente posicionados para matrícula.
